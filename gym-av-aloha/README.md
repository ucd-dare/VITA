## Installation

Clone the repository and install the package in editable mode:

```bash
pip install -e ./gym_av_aloha
```

## Available Environments

* `peg-insertion-v1`
* `cube-transfer-v1`
* `color-cubes-v1`
* `thread-needle-v1`
* `hook-package-v1`
* `pour-test-tube-v1`
* `slot-insertion-v1`
